# super_mario
Building An AI Model to Play Super Mario Game

•	Perception: the AI model perceives the environment through image recognition software such as visual appearances (colors, textures) locations, positions of power-ups , time remaining in the game etc.. 

•	Action: Moving Mario to the right or left, making it jump, speed up, slow down and deciding when to use power-ups. 

•	Goal: To complete the game and reach the highest levels while collecting as many points as possible. 

•	Environment: The environment is a Super Mario game world, including obstacles, enemies, and power-ups.

Programming Language used: Python
